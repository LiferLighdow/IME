# IME (Pure Java, 44KB)

A high-performance, zero-dependency ASCII Input Method for Android.

## 🚀 Highlights
- **Ultra Lightweight:** 44KB APK size.
- **Pure Java:** Built without any Jetpack/Support libraries.
- **Maximum Performance:** Direct Canvas rendering, zero background overhead.
- **Cross-Generation:** Supports Android 5.0 (API 21) up to Android 17+.

## 🛠️ Design Philosophy
Modern mobile software is bloated. `IME` is a "Digital Scalpel" designed for performance-critical environments and minimal AOSP builds (Targeting 100MB RAM standby).
